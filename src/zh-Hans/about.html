什么是Dookbook

简介

Dookbook，Developers' Cookbook（开发者的日常菜谱）的缩写, 它是专为开发者打造的全新知识引擎，为所有开发人员提供了大量有用的代码示例和代码片段。现在包含超过200个菜谱，涵盖从编程语言Python, Web前端技术(HTML, CSS, JavaScript), Java, SQL, Go, PHP等, 到IT基础技术及各类开发库框架, 比如TCP/IP网络, Linux, Nginx, Redis, MySQL, Vue/React, Django/Flask等等.

特性

1、搜索更精准

相比于Google、百度等搜索引擎，直接提供精准的知识结果，不再提供冗长的链接列表，省时省心

2、用知识入股，参与分红

平台会计算修改知识文档等贡献率，共享平台（广告）收益。

3、用开发者的方式提交文档

平台采用 Git + Markdown 的方式，在本地提交文档而无需打开平台网站
