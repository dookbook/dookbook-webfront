    什么是Dookbook?

    更新日期: 2018-10-24

    Dookbook，Developers' Cookbook（开发者的日常菜谱）的缩写.


    简介

    Dookbook是专为开发者打造的全新知识引擎，提供大量有用的代码示例和代码片段。现在包含超过200个菜谱，涵盖从编程语言Python, Web前端技术(HTML, CSS, JavaScript), Java, SQL, Go, PHP等, 到IT基础技术及各类开发库框架, 比如TCP/IP网络, Linux, Nginx, Redis, MySQL, Vue/React, Django/Flask等等.

    特性

    1、搜索精准

    传统搜索引擎Google，百度或者其他只是为您提供了一个冗长的结果列表，需要您不厌其烦得靠自己从链接海洋中找到您想要的内容，(还不一定找得到，原谅我的直接)。完全不同于上述引擎，DookbookTM永远不会让自己成为麻烦制造者。它是一个知识引擎，引入了一套全新的索引机制和算法，使得搜索的结果始终是唯一的，并且它总是幸运的（满足您的需求）。

    2、用知识入股，参与分红

    开发者可以自由地上传新的、更新完善已有的文档。DookbookTM会自动计算每篇知识文档的贡献率，贡献者按各自贡献率百分比共享平台的（广告）收益。

    DookbookTM的盈利模式非常简单，即依靠Google Adsense或百度关键字广告等方式。平台会严格限制广告内容，仅限于与当前文档相关联的广告，比如搜Python相关文档时，只会出现Python培训或者招聘类广告。

    3、更懂开发者

    DookbookTM更懂开发人员，采用 Git + Markdown 的组合。用Markdown编辑文档，用Git提交文档，一切尽如你日常工作提交代码一样。
