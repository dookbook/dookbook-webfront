Dookbook®=Developers' Cookbook，开发者的日常菜谱。

简介

Dookbook® (开发者的日常菜谱)是为开发人员提供海量、实用的代码示例片段，以及编程（工具）技巧的新一代知识引擎。

目前支持的主题包括：

Icon Name Icon Name Icon Name Icon Name
HTML CSS JavaScript Markdown

特性（针对用户）

1. 代码示例片段

Dookbook®为开发人员提供海量、实用的代码示例片段。 它弥补了目前枯燥乏味的官方文档或晦涩难懂的技术规范，与实用、实操、实例之间的鸿沟。

2. Precise Search

传统搜索引擎只是为您提供了一个冗长的结果列表，需要您不厌其烦得靠自己从链接海洋中找到您想要的内容(还不一定找得到，请原谅我的直接)。完全不同于上述引擎， Dookbook®永远不会让自己成为麻烦制造者。它是一个知识引擎，引入了一套全新的索引机制和算法，使得搜索的结果始终是唯一的， 并且它总是幸运的 （满足您的需求）。

特性（针对贡献者）

Tips

文档Git仓库均开放并托管在GitHub上。

1. 更懂开发者

Dookbook®更懂开发人员，采用Git+Markdown的组合。用Markdown编辑文档， 用Git提交文档，一切尽如你日常工作提交代码一样。

2. 知识协作，收益共享

让开发者多一份睡后收益。 开发者可以自由地上传新的、更新完善已有的文档。Dookbook®会自动计算每篇知识文档的贡献率，贡献者按各自贡献率百分比共享平台的（广告）收益。

Dookbook®的盈利模式非常简单，即依靠Google Adsense、百度联盟或自营等线上广告的方式。平台会严格限制广告内容， 仅限于与当前文档相关联的广告，比如搜Python相关文档时，只会出现Python培训或者招聘类广告。
